    >> Div is a block level element ,span is inline element.

       (text, images, header, footer, navigation bar, etc.
       Span is used to group elements for styling purposes (by using the class or id attributes).

5. Difference between block and inline 

            >> A block-level element always starts on a new line, and the browsers 
               automatically add some space (a margin) before and after the element.
               A block-level element always takes up the full width available. Ex : p and div.

               An inline element does not start on a new line.
               An inline element only takes up as much width as necessary. Ex : span

7. What are attributes? 

    >> HTML attributes provide additional information about HTML elements.
       It's some kind of special keyword.
       Attributes usually come in name/value pairs like: name="value"
       Ex : href attribute in a tag,src attribute in img tag.
       Attribute should be written in lowercase only as they are case sensitive.

8. Difference between HTML tags and elements 

    >> Tags are the starting and ending parts of an HTML element. They begin with < symbol and end with > symbol. 
    Whatever written inside < and > are called tags.
    
    Elements enclose the contents in between the tags. They consist of some kind of structure or expression. 
    It generally consists of a start tag, content and an end tag.
