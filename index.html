PART I : CONCEPTS

1. Difference Between Div And Span 

    >> Div is a block level element ,span is inline element.
       Adding div will take some space,margins,but span will not.
       The div tag is used in HTML to make divisions of content on the web page like
       (text, images, header, footer, navigation bar, etc.
       Span is used to group elements for styling purposes (by using the class or id attributes).

2. Difference between class and IDs 

     >> In HTML the only difference between class and id is ids are unique and each element can have only one unqiue id.
        but classes are not unique and we can use same class for multiple elements.
  
        Each page can have only one element with that id but we can use multiple classes on same element.

3. What are self closing tags 

    >> A self closing tags in HTML are the type of HTML tags that need not to be closed manually by its closing tag, which
       means there is no saperate closing tag for it as </tag>.
    
       A few examples of self closing tags are img, input, br, hr etc

4. How many types of headings are there ?How to decide which to use where 

    >> There are Six Types Of Heading h1 to h6. h1 and h2 can be used for title. h3 ,h4 ,h5 least priority over.
       According to our uses we can use them.

5. Difference between block and inline 

    >> A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.
       A block-level element always takes up the full width available. Ex : <p> and <div>

       An inline element does not start on a new line.
       An inline element only takes up as much width as necessary. Ex : <span>

6. Difference between em and strong 

    >> strong and em adds extra semantic meaning to the webpage. They also give a bold and italic style to the text.
       we can use strong to mean that a particular text is strongly emphasized.
       Same goes with em it indicates emphasis and also give italic effect.

7. What are attributes? 

    >> HTML attributes provide additional information about HTML elements.
       It's some kind of special keyword.
       Attributes usually come in name/value pairs like: name="value"
       Ex : href attribute in <a> tag,src attribute in <img> tag.
       Attribute should be written in lowercase only as they are case sensitive.

8. Difference between HTML tags and elements 

    >> Tags are the starting and ending parts of an HTML element. They begin with < symbol and end with > symbol. 
    Whatever written inside < and > are called tags.
    
    Elements enclose the contents in between the tags. They consist of some kind of structure or expression. 
    It generally consists of a start tag, content and an end tag.

9. Explain br tags 

    >> Used To Give Line Break.

10. What are semantic tags? 

    >> A semantic element clearly describes its meaning to both the browser and the developer.
    In HTML there are some semantic elements that can be used to define different parts of a web page.
    Ex : table,form,article,aside,section,footer,header,nav etc.

3. Create a table and explain how to acheive colspan 

    
Code:

    <tr>
    <th>Books</th>
    <th colspan="2">Pens</th>
    <th>Marker</th>
    </tr>
    <tr>
    <td>50</td>
    <td>145</td>
    <td>100</td>
    <td>50</td>
    </tr>

 Explanation : Here we were having 2 types of pen data but only one table heading.
               So We had to use both data in the same table heading(Pen).
               In Table Heading "Pens" we have used colspan to combine the both data in the same heading.
               By using colspan="2"(as we have 2 column data only to span) we were able to merge the columns in the same heading.
